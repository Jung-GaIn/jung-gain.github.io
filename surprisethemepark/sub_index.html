    어드벤처를 나갔다가 다시 들어올 수 있나요?

    롯데월드 어드벤처에서는 파크 입장 후 퇴장하시면 원칙정으로 재입장이 불가능합니다.
    단, 재입장을 도와드리는 몇가지 사항에 대해 안내해 드립니다.

    1. 아이스링크 이용 손님
    2. 저자거리 또는 민속박물관 이용 손님
    3. 의무실 방문 후 간호사 진단에 따라 외부 의료기관 방문하는 경우
    4. 파크 입장 후 매직패스 프리미엄 티켓 구매를 희망하시는 경우

  • 온라인 예매

    다른사람에게 티켓'선물하기'는 어떻게 하나요?

    여러 명의 티켓을 한 분이 모두 구매하셨다면, 선물하기 기능을 이용해 동행하시는 분께 티켓을 전달할 수 있습니다.
    티켓을 선물하는 방법은 아래와 같습니다.

    1. 테마파크 앱을 구동 마이텟에 들어가셔서 보내고자 하는 티켓을 찾아주세요
    2. 티켓 번호 아래 선물하기 버튼을 눌러주세요
    3. 받는 분의 핸드폰 번호를 입력해 주세요.
    4. 받으신 분께서 전송된 메세지의 url을 클릭하면 웹티켓이 생성됩니다.

    ※티켓을 전달할 때는 반드시 앱의 선물하기 기능으로만 보내주세요.

    예매를 했는데 [현장할인불가능]이라는 오류 메시지가 나와요

    현장할인불가능 메시지는 결제 시도한 카드가 제휴카드가 아니거나,
    제휴실적이 충족되지 않았을 경우 뜨는 메시지입니다.
    소지하신 카드가 제휴카드 종류가 맞는지 서프라이즈 테마파크 홈페이지에서 확인 가능하며,
    제휴실적 충족 여부는 해당 카드사를 통해서만 확인이 가능합니다.

    제휴카드 종류 : http://adventure.surprisethemepark.com/kor/price/card/information/list.do

  • 온라인 예매

    [네트워크 상태가 좋지 못하여 정보를 얻어오지 못했습니다]라는 문구가 나와요

    AVG Internet Security 프로그램에 의해 안전결제ISP 서비스가 차단 되었을 경우
    나타날 수 있는 현상입니다.
    이런 경우 아래와 같이 프로그램을 종료하고 다시 결제를 시도하셔야 합니다.
    1. 우측 트레이 아이콘에 있는 AVG Internet Security를 더블클릭해서프로그램을 실행한다.
    2. 메뉴 중 도구 – 고급설정을 클릭한다.
    3. 좌측 메뉴에서 ‘일시적으로AVG보호 사용 안 함’을 선택한 후
    우측 화면에서 ‘일시적으로AVG 보호 사용안함’을 체크하고 적용을 클릭한다.
    4. ‘FireWall 보호 사용 안 함’을 체크하고 확인을 클릭한다.
    5. 확인 버튼을 클릭하면 10분동안 AVG Internet Security 가10분 동안 정지됩니다.

    회원가입, 탈퇴는 어떻게 하나요?

    회원가입과 탈퇴는 서프라이즈 테마파크 홈페이지에서 로그인창에서 가입과 탈퇴를 할수 있습니다.
    현재 서프라이즈 테마파크는 서비스를 편리하고 안전하게 이용 할 수 있는 S.POINT회원제를 운영하고 있습니다.

    서프라이즈 고객만족센터: 1899-0000

  • 기타

    롯데월드 어드벤처 캐스트에 지원하려면 어떻게 해야하나요?

    서프라이즈 테마파크는 즐겁게 일하고 손님에게 최선의 서비스를 제공할 준비가 되어 있는 지원자를 찾고 있습니다.
    서프라이즈 테마파크는 다양한 지원자 분들의 편의와 효율적인 채용관리를 위해 인재채용 홈페이지를 운영해 채용공고를 게시하고 있습니다.

    홈페이지 주소 :
    http://recruit.surprisethemepark.com/

  • 기타

    종합이용권이나 파크이용권, 연간이용권으로 이용 못하는 시설이 있나요?

    종합이용권이나 파크이용권, 연간이용권으로는 대부분의 시설물을 이용하실 수 있으나,

    게임시설(코인 아케이드 시설，서바이벌 게임장, 툼 오브 호러, 거울미러 등) 또는
    유료 지불 시설(환상의숲, 뽀로로파크)은 별도의 요금을 지불하셔야 합니다．

    여러 명의 티켓을 한 분이 모두 구매하셨다면, 선물하기 기능을 이용해 동행하시는 분께 티켓을 전달할 수 있습니다.
    티켓을 선물하는 방법은 아래와 같습니다.

    1. 테마파크 앱을 구동 마이텟에 들어가셔서 보내고자 하는 티켓을 찾아주세요
    2. 티켓 번호 아래 선물하기 버튼을 눌러주세요
    3. 받는 분의 핸드폰 번호를 입력해 주세요.
    4. 받으신 분께서 전송된 메세지의 url을 클릭하면 웹티켓이 생성됩니다.

    ※티켓을 전달할 때는 반드시 앱의 선물하기 기능으로만 보내주세요.

    예매를 했는데 [현장할인불가능]이라는 오류 메시지가 나와요

    제휴실적 충족 여부는 해당 카드사를 통해서만 확인이 가능합니다.

    AVG Internet Security 프로그램에 의해 안전결제ISP 서비스가 차단 되었을 경우
    나타날 수 있는 현상입니다.

    1. 우측 트레이 아이콘에 있는 AVG Internet Security를 더블클릭해서프로그램을 실행한다.
    2. 메뉴 중 도구 – 고급설정을 클릭한다.
    3. 좌측 메뉴에서 ‘일시적으로AVG보호 사용 안 함’을 선택한 후
    우측 화면에서 ‘일시적으로AVG 보호 사용안함’을 체크하고 적용을 클릭한다.
    4. ‘FireWall 보호 사용 안 함’을 체크하고 확인을 클릭한다.
    5. 확인 버튼을 클릭하면 10분동안 AVG Internet Security 가10분 동안 정지됩니다.

    ※유료시설 및 매직패스 대기라인이 없는 시설은 사용이 불가하며, 티켓은 구매 당일만 사용 가능합니다.
